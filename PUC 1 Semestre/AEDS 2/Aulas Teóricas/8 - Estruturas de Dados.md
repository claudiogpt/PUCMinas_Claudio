## Lineares Estáticas:
#### **Pilha** - First In Last Out(Filo)
**Ex**
- Compilador/Sistema Operacional
- Calculadora
- Operações
	-> esta_vazia();
	-> empilhar();
	-> desempilhar();
	-> esta_cheia();
#### **Fila** - First In First Out
- Comprar Ingressos
- Sistemas de Regressão (Testes Automatizados em Engenharia de Software)
- SOs
- Requisição de Bancos de Dados
- Operações
	-> vazia()
	-> enfileirar();
	-> desenfileirar();
	-> cheia()
